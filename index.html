核心功能

覆盖系统优化的方方面面，从 CPU 到显卡，从内存到电源，一站式调校

⚙

CPU 核心调度

为游戏进程绑定专属 CPU 核心，避免线程争抢。支持预设方案保存，开机自动应用，确保每次游戏都在最优核心上运行

⬡

进程优化管理

对标 Process Lasso 的专业级进程管理。实时监控 CPU/内存/IO 占用，调整进程优先级，一键终止无用后台进程

◈

内存深度清理

Mem Reduct 级别的 Standby List 清空，智能回收闲置内存。支持间隔自动清理、阈值触发清理，为游戏腾出更多可用内存

▣

显卡驱动管理

自动检测 NVIDIA / AMD / Intel 显卡型号，智能推荐最稳定的 Game Ready 驱动版本，一键跳转官方下载

◆

显卡参数优化

内置 nvidiaProfileInspector，支持一键导入 .nip 优化预设。精细调校显卡参数，榨干每一点性能潜力

☀

ICC 色彩滤镜

Gamma 校正 + 色温调节 + RGB 通道增益 + 暗部平衡 + ICC 配置文件导入。多种预设一键切换，游戏画面更清晰

⚡

电源计划管理

一键导入 DUN / Intel / X3D 多平台高性能电源方案，释放 CPU 完整性能。支持自定义电源计划并导出

🛠

系统深度优化

禁用遥测服务、Windows Defender、搜索索引等无用后台服务。管理开机启动项，关闭隐私设置，释放系统资源

🎮

游戏场景预加载

提前加载游戏场景资源到内存，减少切换卡顿与加载时间。支持明日之后等游戏的专属优化方案

📊

实时硬件监控

CPU / 内存 / GPU / 磁盘环形仪表盘，实时掌握硬件状态。温度、占用率、频率一目了然
